# App Store Assets

This directory contains assets for Shopify App Store submission.

## Screenshots

- `screenshot-knowledge-base.png` - Knowledge Base page showing URL/sitemap ingestion
- `screenshot-analytics.png` - Analytics dashboard with metrics
- `screenshot-billing.png` - Billing page with pricing plans
- `screenshot-install.png` - Installation/welcome page

## Logos

- `logo.svg` - 200x200px app logo (for App Store listing)
- `app-icon.svg` - 1200x1200px app icon (for App Store icon)
